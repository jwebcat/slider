How It Works

Use Slider to show images, videos and other content.

Slider is a fork of FlexSlider by WooThemes that's been adapted to integrate more closely with the Kraken boilerplate. Some of the default slider functions have also been adjusted to better align with research on slider usability.

Getting Started

Start using Slider in just three easy steps.

  1. Add the slider.css and slider.js files to your site.
  2. Include the HTML below in your markup (obviously replace the images and video scripts with your own content).
  3. Add the script below as either inline JavaScript or in an external file to activate Slider.

HTML

<div class="flexslider">
    <ul class="slides">
        <li><img src="img/cupcake1.jpg"></li>
        <li><img src="img/cupcake2.jpg"></li>
        <li><iframe width="560" height="315" src="http://www.youtube.com/embed/8qjge9U_MiA" frameborder="0" allowfullscreen></iframe></li>
    </ul>
</div>

JavaScript

$(window).load(function() {
    $('.flexslider').flexslider();
});

Fluid Video

If you're including videos in your project, you also need to use FluidVids.js. Make sure it loads prior to Slider.

Optional Advanced Features

A More Usable Carousel

Slider's default settings are designed to make it more usable and accessible:

  • Relative sizing so all elements scale proportionately.
  • Auto-rotation is disabled by default.
  • Navigation arrows are permanently visible.
  • Navigation arrows are icons, not images, for crisp rendering on retina displays.
  • Progressive enhancement. By default, all images and videos are displayed. A small script adds a .js class to the body element, which applies the Slider styles.

For more on carousel usability, read these great articles by Brad Frost, Erik Runyons, the Nielsen Norman Group, and the University of York.

Requires jQuery

Slider requires jQuery 1.4.2 or higher.
