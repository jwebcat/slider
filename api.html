Customizing Slider

Here's an example of a multi-image carousel created using the Slider API.

HTML

<div class="flexslider">
    <ul class="slides">
        <li><img src="img/cupcake1.jpg"></li>
        <li><img src="img/cupcake2.jpg"></li>
        <li><img src="img/cupcake3.jpg"></li>
        ...
    </ul>
</div>

JavaScript

$(window).load(function() {
    $('.flexslider').flexslider({
        itemWidth: 190,
        minItems: 2,
        maxItems: 5,
        prevText: '&lang; <span class="screen-reader">Previous</span>',
        nextText: '&rang; <span class="screen-reader">Next</span>'
    });
});

Advanced Settings

Properties

namespace: "flex-",             //{NEW} String: Prefix string attached to the class of every element generated by the plugin

animationLoop: true,             //Boolean: Should the animation loop? If false, directionNav will received "disable" classes at either end

multipleKeyboard: false,        //{NEW} Boolean: Allow keyboard navigation to affect multiple sliders. Default behavior cuts out keyboard navigation with more than one slider present.
mousewheel: false,              //{UPDATED} Boolean: Requires jquery.mousewheel.js (https://github.com/brandonaaron/jquery-mousewheel) - Allows slider navigating via mousewheel
pausePlay: false,               //Boolean: Create pause/play dynamic element

// Special properties

Screen Reader Text

One of the modification Sliders makes on FlexSlider from WooThemes is the inclusion of the .screen-reader class. This allows you to add helpful description text for the visually impaired without displaying it visually.
